# DeepCTR: Uncertainty-aware Click-through Rate Prediction for Learning to Rank

This repository contains the official implementation of our paper 
"Uncertainty-aware Click-through Rate Prediction for Learning to Rank" 
(Computational Intelligence, 2026).

## Requirements:
- Python 3.8+
- TensorFlow 2.x
- scikit-learn
- numpy, pandas
- networkx (for graph construction)
